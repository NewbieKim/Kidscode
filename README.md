# Kidscode
完成内容：                   
1.项目前期的需求分析及可行性分析，数据库(Sql Server)设计，搭建三层框架，实现总体架构；
2.前端框架采用Bootstrap，同时采用jQury库实习基本的前端UI，后端采用c#编写；
3.实现了基本的登录注册，课程内容分类别展示、评论回复课程、收藏或取消课程；
4.教师管理课程(实现对课程的增删改查)、通过项目打包实现在线scratch编程；
